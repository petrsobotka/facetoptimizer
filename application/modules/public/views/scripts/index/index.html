Welcome!

FacetOptimizer is an Open Source project whose main goal is to provide a software tool for Faceted Navigation analysis and optimization.

If your website incorporates Faceted Classification, doesn't matter whether it is an e-shop or non-profit catalogue, this tool can give you new insight into the way visitors behave on your site and interact with facets.

The tool is built on top of Zend Framework and Doctrine 2. It provides a RESTful API for easy integration with client applications.

Source code is available on GitHub repository.

The application can be used as hosted service or using custom installation.

Requirements

  • PHP 5.3
  • MySQL 5.0 (other DB engines can be used with limitations, see this note)
  • SSL enabled virtual host is strongly recommended
